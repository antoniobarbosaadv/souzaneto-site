## FASE 1 — A Lívia nasce (grátis, valida antes de pagar)
- [ ] 🧑‍💼 HeyGen → **Avatar → Avatares fotográficos → Gerar com IA** → colar o prompt da Lívia.

- [ ] 🧑‍💼 Criar o avatar e **nomear "Lívia"**.
- [ ] 🧑‍💼 Gerar **1 vídeo-teste curto** (com a voz nativa do HeyGen) — mesmo com marca d'água.
- [ ] 🧑‍💼+🤖 **Você aprova rosto + voz.** (Não gostou? Ajustamos o prompt — custo zero.)

## FASE 2 — Assinar o que precisa (só depois de aprovar a Lívia)

- [ ] 🧑‍💼 **Metricool** — publicação (conectar IG + TikTok do massificado).

- [ ] 🧑‍💼 HeyGen → **Painel da API** → copiar a **API key**.
- [ ] 🧑‍💼 Metricool → API/conexão → copiar o token.
- [ ] 🧑‍💼 Guardar tudo no Bitwarden.

- [ ] 🧑‍💼 No Make: **Importar blueprint** → **colar as chaves** → conectar o Drive.
- [ ] 🧑‍💼 Rodar **1 teste** → me dizer o que deu → 🤖 eu corrijo o arquivo (uma vez).

## FASE 5 — Ligar a esteira (e nunca mais tocar)
